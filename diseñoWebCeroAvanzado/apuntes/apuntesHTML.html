Etiquetas nuevas en HTML5

Antes todo eran divs con identificadores, como header, nav, content, aside, etc. Actualmente ya existen etiquetas especificas para algunas secciones.

  • header
  • nav
  • section
  • article
  • aside
  • footer

Etiquetas obsoletas

En su mayoria son etiquetas de estilo que ahora debe de hacerse con CSS

Las imagenes ahora se insertan con figure, ya no se utiliza img

  • acronym.- Etiqueta que define un acrónimo, palabras clave que dan informacion a los navegadores o motores de busqueca. (Utilizar Etiqueta abbr)
  • applet.- Etiqieta que sirve para incrustar un applet en nuestra pagina. (Utilizar object)
  • basefont.- Etiqieta que establece el valor predeterminado del formato de la fuente de todo el texto del documento. (Utilizar CSS)
  • big.- Etiqueta que hace destacar el texto haciendo mas grande respecto del que tienen al lado. (Utilizar CSS)
  • center.- Etiqieta que alinea el texto contenido en el centro de la pagina. (Utilizar CSS)
  • dir.- Etiqueta que defini ina lista de directorios. (Utilizar ul)
  • font.- Etiqueta que establece el formato de la fuente de algun elemento en concreto. (Utilizar CSS)
  • frame.- Etiqueta que defini marcos o paginas mas pequeñas dentro de nuestra pagina.
  • frameset.- Etiqueta que define el conjunto de marcos paginas mas pequeñas dentro de nuestra pagina
  • isindex.- Etiqueta que crea un control de entrada de texto. (Utilizar input)
  • noframes.- Etiqueta que contiene el contenido a mostrar en caso de que el navegador no soporte Frames
  • strike.- Etiqueta que muestra el texto tachado. (Utilizar del)(Utilizar CSS
  • tt.- Etiqieta que establece texto de teletipo. (Utilizar CSS)
